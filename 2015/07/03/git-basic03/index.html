分支

所创建的分支，都是基于当前分支创建的，也可以指定分支创建。


Fast-forward表示当前合并是快进模式，速度快，但删除分支后，会丢失分支信息。

建议禁用Fast forward模式，git会在merge时产生一个新的commit，从分支历史上就可以看出分支信息。
